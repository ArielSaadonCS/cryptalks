During this assignment, I used AI tools as assistants throughout the development process.

First, i used ChatGPT to discuss the product vision and architecture before implementation. We analyzed the task requirements, defined the product as a personalized daily crypto briefing dashboard, decided to use a Dockerized modular architecture instead of microservices, planned the main modules, and clarified the user flow: authentication, onboarding preferences, dashboard, and feedback.

After the vision and architecture, i handed the plan to a separate chat gpt for prompt engineering help create structured step by step prompts for Claude Code. I then used Claude Code to implement the project gradually, module by module, while reviewing the results and keeping the implementation aligned with the original scope.

After the core project was working, the frontend was functional but basic. I zipped the project, uploaded it to Lovable, and asked it to improve the frontend design and make the UI feel fresher and more polished. Then I cloned Lovable’s result into VS Code and used Claude Code again to integrate the improved frontend carefully, while avoiding unnecessary complexity or features that were not relevant to the assignment.

Overall, i treated the tools as development assistants, while keeping the product direction, technical decisions, and final scope under my control.
